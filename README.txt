This folder contains code used to generate the figures in the report
"Extracting Neutrino Oscillation Parameters Using a Negative Log-Likelihood Fit".

-------------------------------------------------------------------------------------------
|                  Files                  |                  Description                  |
-------------------------------------------------------------------------------------------
|main.py                                  |Run this file to generate all the plots regard-|
|                                         |ing minimizing the NLL.                        |
-------------------------------------------------------------------------------------------
|validation.py                            |Run this file to generate all the plots regard-|
|                                         |ing minimizing the NLL, and also plots to veri-|
|                                         |fy that all the functions are working correctly|
|                                         |.                                              |
-------------------------------------------------------------------------------------------
|secant_test.py                           |This is a simple script testing the secant met-|
|                                         |hod to solve equations.                        |
-------------------------------------------------------------------------------------------
|tools.py                                 |This file contains additional algorithms not r-|
|                                         |elated to minimization. Most of the functions  |
|                                         |are not used.                                  |
-------------------------------------------------------------------------------------------
|neutrino_energy.csv                      |Data file.                                     |
-------------------------------------------------------------------------------------------

Note: For IDEs that support code cells, main.py and validation.py have been separated into
cells using #%%.
